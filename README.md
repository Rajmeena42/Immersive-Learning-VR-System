# My Unity Project

A standard Unity 6 project template.

## Project Specifications

- **Unity Version:** `6000.4.3f1`
- **Platform:** PC, Mac & Linux Standalone (Default)
- **Render Pipeline:** Built-in / Standard Render Pipeline (Default)

## Folder Structure

- `/Assets`: Contains all game scenes, scripts, prefabs, audio, and visual assets.
  - `/Assets/Scenes`: Contains the default `SampleScene.unity`.
- `/Packages`: Lists all external package dependencies (configured in `manifest.json`).
- `/ProjectSettings`: Project configurations for inputs, physics, audio, tags, quality levels, etc.

## How to Run

1. Open **Unity Hub**.
2. Click **Add** -> **Add project from disk**.
3. Select this repository folder (`My project`).
4. Click on the project to open it in **Unity Editor 6000.4.3f1**.
5. Press the **Play (▶️)** button at the top of the editor to run the game in the editor.

## Development Notes

This repository uses a `.gitignore` tailored specifically for Unity projects to exclude auto-generated files like the `/Library`, `/Temp`, `/Obj`, and `.vs` folders. Only source files and project settings are tracked.
